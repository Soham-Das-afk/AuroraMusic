AuroraMusic v3.2.35
====================

Release Date: 2025-10-14

Summary
-------
Multi-architecture Docker publishing added (linux/amd64 + linux/arm64) and documentation/version bump.

Changes
-------
### Added
- Multi-arch Docker build in release workflow (`platforms: linux/amd64,linux/arm64`).
- Manifest inspection step to verify pushed platforms.

### Changed
- Bumped version to 3.2.35 across config, README, and admin cog fallback.
- README Docker section now notes multi-architecture images starting with v3.2.35.

### Notes
- Tag this commit as `v3.2.35` to trigger the new multi-arch build & release pipeline.
- Both `sohamdas103/auroramusic:3.2.35` and `:latest` will publish as a multi-platform manifest.

Upgrade Instructions
--------------------
Docker (Compose):
```
docker compose pull
docker compose up -d --pull always
```

Direct Docker:
```
docker pull sohamdas103/auroramusic:3.2.35
```

From source:
```
git pull
python src/bot.py
```
